# 📁 assets/screenshots/

Coloca aquí los screenshots de tus proyectos.

## Convención de nombres

| Archivo               | Proyecto              |
|-----------------------|-----------------------|
| `project1.png`        | Primer proyecto       |
| `project2.png`        | Segundo proyecto      |
| `project3.png`        | Tercer proyecto       |

## Especificaciones
- Formato: **PNG o JPG**
- Tamaño recomendado: **1280×720 px** (16:9)
- Se muestran en `ProjectDetail.jsx` dentro del área de demo

## Uso
Actualiza `src/data/projects.js` con la ruta del screenshot:
```js
screenshot: '/assets/screenshots/project1.png'
```
