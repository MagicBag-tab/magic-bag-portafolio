# 📁 assets/flowers/

Coloca aquí los elementos decorativos de la pantalla de proyectos.

## Archivos esperados

| Archivo           | Descripción                              |
|-------------------|------------------------------------------|
| `flower-1.png`    | Flor decorativa 1 (rosa / lavanda)       |
| `flower-2.png`    | Flor decorativa 2                        |
| `flower-3.png`    | Flor decorativa 3                        |
| `petal.png`       | Pétalo individual (para animaciones)     |
| `star.png`        | Estrella decorativa                      |

## Especificaciones
- Formato: **PNG con fondo transparente**
- Tamaño recomendado: **64–128 px** de alto
- Estilo: pixel art o acuarela, paleta rosa/lavanda

## Uso en ProjectsScreen
Las flores se posicionan aleatoriamente como fondo decorativo en `ProjectsScreen.jsx`.
Para usar imágenes reales en vez de emojis, actualiza el componente para usar `<img>` con estas rutas.
